## 安装

### 构建过程

1. 确保已安装所有依赖库.

_不同发行版的软件包名称可能不同，如果您的发行版提供了日历，请检查发行版提供的打包脚本。_

如果你使用的是 [Deepin](https://distrowatch.com/table.php?distribution=deepin) 或者其它提供了日历的发行版:

``` shell
$ sudo apt-get build-dep dde-calendar
```

2. 构建:

```
$ cd dde-calendar
$ mkdir Build
$ cd Build
$ cmake ../
$ make
```

3. 安装:

```
$ sudo make install
```

可执行程序为 `/usr/bin/dde-calendar

## 使用

命令行执行 `dde-calendar`
